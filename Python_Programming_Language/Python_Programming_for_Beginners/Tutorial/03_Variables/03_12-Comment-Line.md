# :zero::three: Variables :bookmark:
## :one::two: Comment Line
> ### Description 
> Python da yorum satırıları iki şekildedir.
> * Tek satır yorumlar için # karakateri ile başlanır.
> * Çok satır yorum için ise """...""" üç tane çift tırnak içinde yazılır
> 
> ``` python
> # Bu satır bir yorum satırıdır. Tek satırda yazılır.
> # Bu satır da bir yorum satırdır. Yorum satırları ekran çıktı olarak görülmez
> # Python bu yazılanları göz ardı eder. Bu yorumlar yazılımcılar için açıklama notlarıdır
> print('Merhaba beni ekranda görüyorsunuz')
> print('Ancak kodun içinde yazılan yorumları göremiyorsunu')
> """
> Bu satırlar da yorum satırıdır. Ancak diğerlerinden farklı olarak çok satır desteği vardır
> Yani şuan uzun bir yorum yazabiliriz. Bazı kod bloklarının üzerinden yazarın adının soyadnın
> kodun yazıldığı zamanı belirten küçük bilgi satırları olur. Bunları bu şekilde yazabiliriz.
> """
> 
> print('Benden önce yazılan uzun bir yorum satırı vardır. Ancak siz bunu göremiyorsunuz')
> 
> text = "Bu bir text mesaj değeridir. Bu satırın hemen soununda bir yorum vardır." # bu da bir yorum satırıdır.
> text += " Yorum satırından sonra bu yazıyı ekledim."
> 
> print(text)
> ```
